Requerimientos

El programa pedirá su número telefónico(celular o fijo); si no está registrado se le pedirá su nombre, se le generara una contraseña y se le dará a elegir entre una lista de preguntas y se le pedirá su respectiva respuesta para poder recuperar su contraseña en caso de que se olvide y se finalizara el programa habiendo creado un archivo con sus datos(nombre, número de teléfono, contraseña, respuesta secreta).

Si es que ya se encuentra registrado lo que se le pedirá al usuario será su contraseña; si es incorrecta por 3 veces se le pedirá su respuesta a la pregunta secreta de no recordarla, registrarse nuevamente. Una vez identificado se le pedirá el tipo de producto que desea, después se le dará una lista de los productos del tipo elegido anteriormente, de la cual deberá elegir uno de ellos para luego mostrar las tiendas o bodegas (coordenadas) de las que quiere adquirir el producto.

El programa se le pedirá su ubicación (coordenadas); con esta coordenada se le generara las 3 tiendas más cercanas ordenadas según su distancia, luego se le muestra las características de la tiendas o bodegas(tiempo de llegada y calificación). Luego se muestran las tiendas o bodegas ordenadas según su tiempo de llegada. Después se muestran las tiendas o bodegas ordenadas según su calificación. Se le da a elegir al usuario la bodega que le parezca más conveniente. Finalmente se le mostrará al usuario un mensaje de confirmación.

Muchas Gracias
